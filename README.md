Welcome to the Arcade Game - SuckBugs.

Here the rules:

How to Start Game:


1. Download code

2. Open index.html

3. Enjoy the game.




How to play:

1. Use arrow keys to control our hero.

2. Get to the water line and you will get 100 points.

3. Don't touch the bugs or you will start again and will lose 10 points.
